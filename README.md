## Deskripsi
Aplikasi laundry berbasis Laravel untuk mengelola pelanggan dan transaksi laundry.

## Fitur
- Login
- Register
- Dashboard
- Manajemen Customer
- Manajemen Order

## Teknologi
- Laravel
- PHP
- MySQL
- Bootstrap

## Instalasi

composer install

cp .env.example .env

php artisan key:generate

php artisan migrate

php artisan serve
