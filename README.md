## Developer
Bagus Hermawan

## Deskripsi
Ardian Laundry adalah aplikasi manajemen laundry berbasis Laravel yang digunakan untuk mengelola pelanggan dan transaksi laundry.

## Fitur
- Login & Register
- Dashboard Admin
- Manajemen Customer
- Manajemen Order
- CRUD Data Laundry

## Teknologi
- Laravel
- PHP
- MySQL
- Bootstrap

## Cara Menjalankan

```bash
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
```
